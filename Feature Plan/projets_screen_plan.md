# Projets
Présenter le portfolio des réalisations de MAC pour démontrer expertise et qualité.

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Page Header Section (Full-width)
- Content Container (Below header):
  - Filters and Search
  - Projects Grid
  - Load More
- Footer (Full-width)

## Top Navigation Bar
- Logo MAC
- Menu de navigation principal (Accueil, Services, Projets, Blog, À Propos, Contact)
- Bouton "Demander un Devis"
- Sélecteur de langue (FR/EN)

## Page Header Section
- Titre de page "Nos Réalisations"
- Sous-titre valorisant l'expertise et la qualité
- Image de fond d'un projet emblématique

## Filters and Search
- Barre de recherche par nom de projet
- Filtres par type de projet (Tous, Résidentiel, Commercial, Industriel, Forage)
- Filtres par localisation (pays/régions)
- Filtre par année de réalisation
- Nombre de résultats affichés

## Projects Grid
- Cartes de projets en grille responsive (3 colonnes desktop, 2 tablette, 1 mobile)
- Chaque carte contient:
  - Photo principale du projet
  - Badge du type de projet
  - Nom du projet
  - Localisation
  - Année de réalisation
  - Extrait de description courte
  - Bouton "Voir les détails"
- Affichage de 12 projets variés par page

## Load More
- Bouton "Charger plus de projets"
- Indication du nombre total de projets disponibles

## Footer
- Logo et description courte de MAC
- Liens rapides (Services, Projets, Blog, À Propos, Contact)
- Coordonnées complètes (adresse, téléphone, email)
- Liens réseaux sociaux
- Mentions légales et politique de confidentialité
- Copyright
